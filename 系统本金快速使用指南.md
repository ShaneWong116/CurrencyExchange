# 系统本金快速使用指南

## 🎯 什么是系统本金？

系统本金是系统的**总资本金额**（以港币计算），它与利润挂钩：

```
💰 新本金 = 旧本金 + 本次利润 - 其他支出
```

## 📍 在哪里调整本金？

### 访问路径
1. 登录后台管理系统：`http://your-domain/admin`
2. 左侧菜单找到 **"财务管理"** 分组
3. 点击 **"系统本金"**

![本金管理位置](https://via.placeholder.com/800x400?text=财务管理+→+系统本金)

## 🚀 首次使用：初始化本金

### 第一步：进入本金管理
- 后台管理 → 财务管理 → **系统本金**

### 第二步：调整本金
1. 点击右上角 **"调整本金"** 按钮
2. 填写表单：
   ```
   调整后本金: 100000  (输入初始本金金额，单位：港币)
   调整类型: 手动调整
   调整原因: 初始化系统本金
   ```
3. 点击 **"保存"**

### 完成！
- 页面顶部会显示：💰 **当前系统本金: HK$ 100,000.00**
- 本金调整记录已自动创建

## 📊 日常使用场景

### 场景1：查看当前本金
- 进入 **财务管理 → 系统本金**
- 页面顶部绿色标签显示当前本金

### 场景2：手动调整本金
**适用情况**：
- 增资/减资
- 本金校正
- 资本重组

**操作步骤**：
1. 点击 **"调整本金"**
2. 输入调整后的金额（如增资 50,000，输入 150000）
3. 填写调整原因（如："增资 50,000 港币"）
4. 保存

### 场景3：结算后自动更新
执行结算操作时，**系统会自动更新本金**，无需手动操作！

**例如**：
```
当前本金: 100,000 HKD
本次利润: 5,000 HKD
其他支出: 500 HKD

结算后自动更新：
新本金 = 100,000 + 5,000 - 500 = 104,500 HKD
```

系统会自动创建一条"结算调整"类型的记录。

## 📋 本金记录表格说明

### 表格列说明

| 列名 | 说明 | 示例 |
|------|------|------|
| **调整前本金** | 调整前的本金金额 | HK$ 100,000.00 |
| **调整金额** | 增加或减少的金额 | +HK$ 5,000.00 |
| **调整后本金** | 调整后的新本金（加粗显示） | **HK$ 105,000.00** |
| **调整类型** | 手动/结算/系统 | 🟡 手动调整 / 🟢 结算调整 |
| **关联结算** | 如果是结算触发，显示结算编号 | #001 |
| **操作人** | 谁进行的调整 | admin |
| **调整原因** | 为什么调整 | 初始化系统本金 |
| **调整时间** | 调整的时间 | 2025-10-27 14:30:00 |

### 颜色标识

- 🟢 **绿色**（正数）：本金增加
- 🔴 **红色**（负数）：本金减少
- 🟡 **黄色标签**：手动调整
- 🟢 **绿色标签**：结算调整
- 🔵 **蓝色标签**：系统调整

## 🆘 常见问题

### Q1: 当前本金显示为 0 怎么办？
**A**: 说明还未初始化本金，按照"首次使用"步骤设置初始本金即可。

### Q2: 本金调整记录能删除吗？
**A**: 不能。所有本金调整记录都是永久保存的，用于审计和追踪。

### Q3: 结算后本金会自动更新吗？
**A**: 是的！每次执行结算操作，系统会自动计算并更新本金。

### Q4: 如何查看本金历史变化？
**A**: 在本金管理页面，所有调整记录按时间倒序显示，可以清楚看到每次变化。

### Q5: 本金和余额有什么区别？
**A**: 
- **本金**：系统总资本（港币），用于利润累积
- **余额**：各渠道的日常资金（人民币/港币）

两者是独立管理的。
